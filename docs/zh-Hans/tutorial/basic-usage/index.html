基本使用

本指南涵盖使用 Omx 的基本快捷键、命令和菜单选项。

键盘快捷键​

快捷键操作
Escape打开菜单（空闲时）/ 中断生成（加载时）
Ctrl+C按两次退出
Ctrl+A将光标移到行首
Ctrl+E将光标移到行尾
Ctrl+L清空输入框
Shift+Enter插入换行（多行输入）
Ctrl+J插入换行（TUI）
Up/Down在多行输入中移动光标 / 浏览输入历史

Slash 命令​

在聊天中直接输入这些命令：

命令描述
/clear开始全新的对话
/status显示会话、模型和配置状态
/compact手动压缩上下文并重新开始
/rewind回退到之前的消息
/model切换到不同的模型
/session加载之前的会话
/git-commit为暂存的更改生成提交消息并提交
/exit退出 Omx

当你开始输入 / 时，会出现一个匹配命令的选择器。使用方向键导航，Enter 键选择。

菜单​

按 Escape 打开菜单。在这里你可以：

  • 切换到不同的模型
  • 加载之前的会话
  • 回退到之前的消息
  • 切换工作流预设
  • 管理你的模型列表
  • 更改偏好设置

工作流预设​

Omx 支持五种工作流预设：

  • Normal - 基础工具，需要手动协调
  • Specialist（默认）- 自主运行的智能体工具。参见专家模式
  • Artist - 主要以图像响应，适合视觉设计。参见艺术家模式
  • Explorer - 先联网搜索再回答问题。参见探索者模式
  • Assistant - 连接你的应用（浏览器、Office 等）的个人助手，直接读取、编辑和创建内容

偏好设置​

偏好设置子菜单让你切换：

  • 自动打开 Diff 面板 - 自动在 Diff 面板中显示文件更改（Web/VS Code）
  • 缓存时长 - Anthropic 缓存提示的时间（5 分钟或 1 小时）
  • 完成通知 - 响应超过一分钟时发送通知
  • 上下文编辑 - 压缩时精简上下文
  • 跨会话记忆 - 跨会话记住关键点
  • 扩展思考 - 为复杂任务启用更深入的推理
  • IDE 上下文 - 包含来自你 IDE 的上下文
  • 服务端压缩 - 让服务端处理上下文压缩，而非客户端
  • 流式输出 - 在生成时显示响应
  • 思考块展开 - 思考块默认展开或折叠

使用方向键导航，Enter 选择，Escape 返回。
