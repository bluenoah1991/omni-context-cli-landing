# 移动端访问

随时随地用手机或平板写代码。Omx 的 Web 客户端在移动设备上运行流畅，让你在离开电脑时也能查看代码、进行编辑、与 AI 对话。

## 启动服务器

首先，以服务器模式启动 Omx 并开启网络访问：

```bash
omx --serve --host 0.0.0.0 --port 5281
```

`--host 0.0.0.0` 参数允许其他设备通过网络连接，而不仅仅是本机访问。

## 连接设备

手机和电脑需要在同一网络下。如果不在同一局域网（比如手机用移动数据），你需要使用 VPN 或组网软件。

### 使用贝锐蒲公英（推荐）

[贝锐蒲公英](https://pgy.oray.com/) 可以在不同网络的设备之间创建安全的虚拟局域网。

1. 在电脑和手机上安装蒲公英客户端
2. 使用同一账号登录两台设备
3. 在蒲公英管理后台将两台设备加入同一网络
4. 查看电脑的蒲公英虚拟 IP（通常是 `172.x.x.x`）
5. 使用该 IP 从手机连接

蒲公英免费版支持 3 台设备组网，足够个人使用。

### 使用局域网

如果两台设备在同一 WiFi 下：

1. 查找电脑的本地 IP 地址：
   - **Windows**: 运行 `ipconfig`，查找 IPv4 地址
   - **macOS/Linux**: 运行 `ifconfig` 或 `ip addr`
2. IP 地址通常类似 `192.168.x.x` 或 `10.0.x.x`

## 从手机访问

打开手机浏览器，访问：

```
http://<你的IP>:5281
```

例如：`http://172.16.0.1:5281` 或 `http://192.168.1.100:5281`

Web 客户端将以完整功能加载。

## 添加到主屏幕

为了最佳体验，将 Omx 作为 PWA（渐进式 Web 应用）添加到主屏幕：

### iOS (Safari)

1. 在 Safari 中打开 Omx Web 客户端
2. 点击分享按钮（带箭头的方框）
3. 向下滚动，点击 **添加到主屏幕**
4. 命名为 "Omx"，点击 **添加**

### Android (Chrome)

1. 在 Chrome 中打开 Omx Web 客户端
2. 点击右上角三点菜单
3. 点击 **添加到主屏幕** 或 **安装应用**
4. 确认安装

安装后，Omx 会以应用图标的形式出现在主屏幕上。打开时是独立窗口，没有浏览器界面，就像原生应用一样。

## 移动端使用技巧

- **竖屏模式** 适合聊天为主的任务
- **横屏模式** 有更多空间查看代码
- 网络较慢时，使用 **模型切换器** 选择响应更快的模型
- **工具审批模式** (`--approve-write`) 在你想要先审查更改再执行时很有用
- 界面针对触控优化，点击区域更大

## 保持服务器运行

要在关闭终端后保持 Omx 运行：

### 使用进程管理器

```bash
# 使用 pm2（安装方式：npm install -g pm2）
pm2 start "omx --serve --host 0.0.0.0 --port 5281" --name omx

# 查看状态
pm2 status

# 停止
pm2 stop omx
```

### 使用 Screen 或 Tmux

```bash
# 启动 screen 会话
screen -S omx

# 运行 Omx
omx --serve --host 0.0.0.0 --port 5281

# 按 Ctrl+A 然后按 D 分离会话
# 之后用 screen -r omx 重新连接
```

现在只要服务器在运行，你就可以随时从手机访问 Omx 了。
