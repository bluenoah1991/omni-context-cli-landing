# 艺术家模式

艺术家模式将 Omx 转变为视觉优先的助手。它不再以文字响应，而是生成图像来回答你的问题和探索想法。

## 工作原理

当艺术家模式激活时，Omx 会以视觉方式解读请求。提出问题，它会以图像回应。描述某个事物，它会展示给你看。想要探索想法？它会将其可视化。

文字响应会保持最少 - 仅用于说明正在创建的内容，或在请求不明确时询问更多细节。

## 适用场景

艺术家模式适合：

- 视觉设计探索
- 概念艺术和创意构思
- 创建插图和图形
- 将抽象想法可视化
- 快速原型和草图

## 启用艺术家模式

通过菜单切换到艺术家模式：

1. 按 `Escape` 打开菜单
2. 选择 **Switch workflow preset**
3. 选择 **Artist**

你也可以使用 `/status` 查看当前的工作流预设。

## 使用技巧

- 详细描述构图、风格、色调、光线和氛围
- Omx 会艺术性地解读请求，而非字面理解，以创造更有感染力的视觉效果
- 如果请求确实无法以视觉方式回答，Omx 会简要说明并提供视觉替代方案
