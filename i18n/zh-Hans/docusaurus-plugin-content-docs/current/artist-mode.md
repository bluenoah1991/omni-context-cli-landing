# 艺术家模式

艺术家模式将 Omx 转变为视觉优先的助手。在模型支持时会优先用图像响应。

## 工作原理

当艺术家模式激活时，Omx 会以视觉方式解读请求，在可用时输出图像。文字响应会保持最少，除非需要澄清。

## 适用场景

艺术家模式适合：

- 视觉设计探索
- 概念艺术和创意构思
- 创建插图和图形
- 将抽象想法可视化
- 快速原型和草图

## 启用艺术家模式

通过菜单切换到艺术家模式：

1. 按 `Escape` 打开菜单
2. 选择 **Switch workflow preset**
3. 选择 **Artist**

你也可以使用 `/status` 查看当前的工作流预设。

## 使用技巧

- 详细描述构图、风格、色调、光线和氛围
- Omx 会艺术性地解读请求，而非字面理解，以创造更有感染力的视觉效果
- 如果请求确实无法以视觉方式回答，Omx 会简要说明并提供视觉替代方案
