# 探索者模式

探索者模式将 Omx 转变为研究优先的助手。它不会直接回答，而是先搜索网络以获取最新、准确的信息。

## 工作原理

当探索者模式激活时，Omx 会像研究员一样思考。提出问题，它会发起多次网络搜索来收集全面的信息。想要了解某个主题？它会从多个角度进行探索。需要事实？它会用最新数据进行验证。

文字响应会综合从网络发现的内容，并在相关时引用来源。

## 适用场景

探索者模式适合：

- 研究时事新闻
- 查找最新文档
- 验证事实和数据
- 了解新技术
- 比较产品或服务

## 启用探索者模式

通过菜单切换到探索者模式：

1. 按 `Escape` 打开菜单
2. 选择 **Switch workflow preset**
3. 选择 **Explorer**

你也可以使用 `/status` 查看当前的工作流预设。

## 使用技巧

- Omx 每个问题至少发起 5 次搜索，确保全面覆盖
- 如果搜索没有返回有用结果，它会重新组织或拆分查询
- 如果请求确实无法从网络搜索中受益，Omx 会简要说明并直接回答
