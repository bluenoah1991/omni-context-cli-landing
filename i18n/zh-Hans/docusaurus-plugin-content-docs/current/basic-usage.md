# 基本使用

本指南涵盖使用 Omx 的基本快捷键、命令和菜单选项。

## 键盘快捷键

| 快捷键 | 操作 |
|----------|--------|
| Escape | 打开菜单（空闲时）/ 中断生成（加载时）|
| Ctrl+C | 按两次退出 |
| Ctrl+A | 将光标移到行首 |
| Ctrl+E | 将光标移到行尾 |
| Shift+Enter | 插入换行（多行输入）|
| Up/Down | 在多行输入中移动光标 / 浏览消息历史 |

## Slash 命令

在聊天中直接输入这些命令：

| 命令 | 描述 |
|---------|-------------|
| `/clear` | 开始全新的对话 |
| `/status` | 显示会话、模型和配置状态 |
| `/compact` | 手动压缩上下文并重新开始 |
| `/rewind` | 回退到之前的消息 |
| `/model` | 切换到不同的模型 |
| `/session` | 加载之前的会话 |
| `/git-commit` | 为暂存的更改生成提交消息并提交 |
| `/exit` | 退出 Omx |

当你开始输入 `/` 时，会出现一个匹配命令的选择器。使用方向键导航，Enter 键选择。

## 菜单

按 Escape 打开菜单。在这里你可以：

- 切换到不同的模型
- 加载之前的会话
- 回退到之前的消息
- 切换专家模式
- 管理你的模型列表
- 更改偏好设置

### 偏好设置

偏好设置子菜单让你切换：

- **扩展思考** - 为复杂任务启用更深入的推理
- **流式输出** - 在生成时显示响应
- **IDE 上下文** - 包含来自你 IDE 的上下文
- **Playbook 记忆** - 跨会话记住关键点
- **Anthropic 缓存 TTL** - 设置 Anthropic 缓存你的提示的时间（5 分钟或 1 小时）

使用方向键导航，Enter 选择，Escape 返回。
