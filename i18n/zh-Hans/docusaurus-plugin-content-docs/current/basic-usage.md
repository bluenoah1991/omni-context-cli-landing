# 基本用法

这份指南介绍你在 OmniContext CLI 里最常用的快捷键、命令和菜单路径。

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| Escape | 空闲时打开菜单，或在响应流式输出时中断生成 |
| Ctrl+C | 连按两次退出 |
| Ctrl+A | 将光标移到行首 |
| Ctrl+E | 将光标移到行尾 |
| Ctrl+L | 清空当前输入和附件 |
| Shift+Enter | 插入换行 |
| Ctrl+J | 插入换行 |
| Up/Down | 在多行输入中移动，或浏览输入历史 |

## 斜杠命令

直接在聊天框中输入这些命令：

| 命令 | 描述 |
|------|------|
| /clear | 开始一段全新的对话 |
| /status | 显示当前会话、模型、工作流和偏好状态 |
| /compact | 手动压缩当前上下文 |
| /rewind | 回退到更早的一条消息 |
| /model | 切换到其他模型 |
| /session | 加载之前的会话 |
| /git-commit | 为已暂存的变更生成提交信息并提交 |
| /exit | 退出 OmniContext CLI |

当你输入 `/` 时，OmniContext CLI 会弹出一个匹配命令的选择器。技能也会出现在这里，所以你可以不用离开键盘就加载可复用的提示指令。

## 菜单

按 `Escape` 打开菜单。你可以在这里：

- 切换模型
- 加载更早的会话
- 回退到某条历史消息
- 切换工作流
- 管理模型列表
- 修改偏好设置

### 工作流预设

OmniContext CLI 内置两个工作流：

- **Programming** - 默认的编程工作流，包含基础工具、内置智能体工具、自定义 Agent、MCP 和远程工具
- **General** - 更通用的助手工作流，适合文档、表格、演示文稿和日常任务

如果你使用桌面应用，它还会额外安装一个给 Chrome 侧边栏使用的 **Browser** 工作流。

自定义工作流放在 `~/.omx/workflows/` 或 `.omx/workflows/` 中。完整格式见 [工作流](/tutorial/workflows)。

### 偏好设置

偏好设置菜单可以切换或调整：

- **缓存时长** - `5m` 适合快速迭代，`1h` 适合更长时间的提示缓存
- **完成通知** - 对较慢的回复发送桌面通知
- **上下文编辑** - 裁剪旧的工具调用和推理块，减小上下文体积
- **Diff 视图** - OmniContext CLI 编辑文件时显示内联 diff
- **跨会话记忆** - 在会话之间保留有用的项目知识
- **扩展思考** - 为更复杂的任务开启更深的推理
- **IDE 上下文** - 把编辑器中的上下文带进对话
- **服务器压缩** - 让服务器而不是客户端来处理压缩
- **流式输出** - 按 token 实时渲染回复

在主菜单里，你还可以切换：

- **响应语言** - 强制回复为英文或简体中文，或者保持自动
- **颜色主题** - 选择内置配色方案

用方向键移动，按 `Enter` 确认，按 `Escape` 返回上一级。
