# Chrome 扩展

OmniContext Chrome 扩展为任何网页添加 AI 侧边栏，连接到你运行中的 omx 实例。

## 安装

1. 启动桌面应用或运行 `omx --serve`
2. 从桌面应用的资源文件夹加载浏览器扩展
3. 扩展自动连接到本地 omx 服务器

## 功能

- **页面上下文** - 扩展可以读取当前页面内容
- **总结** - 总结文章、文档或任何网页
- **数据提取** - 从网页中提取结构化数据
- **浏览器自动化** - 通过自然语言自动化浏览器任务

## 使用

点击扩展图标打开侧边栏面板。你可以在浏览时与 omx 聊天——它可以访问当前页面内容并与浏览器交互。

## 连接

扩展连接到你的 omx 服务器（默认：`http://localhost:5281`）。使用扩展前确保 omx 在 serve 模式下运行。
