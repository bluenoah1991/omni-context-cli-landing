# 快速开始

OmniContext CLI 是专为开发者设计的强大智能体工具，帮助您自主管理复杂的开发任务。

## 安装

```bash
npm install -g omni-context-cli && omx
```

## 快速入门

安装完成后，只需运行：

```bash
omx
```

这将启动交互式智能体界面。

## 功能特性

- **专家模式**：针对特定任务的专注智能体
- **零遥测**：隐私优先设计，不收集数据
- **ACE**：智能体式上下文工程
- **多模型**：支持多种 LLM 提供商
- **智能体技能**：可扩展的智能体能力
