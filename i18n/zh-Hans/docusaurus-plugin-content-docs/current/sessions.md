# 会话管理

omx 自动保存对话会话，支持恢复、回退和切换。

## 自动保存

会话在聊天过程中自动保存。每个会话记录完整的对话历史、使用的模型和 Token 用量。

## 加载会话

从命令行恢复上次会话：

```bash
omx --continue
```

或通过菜单或斜杠命令加载任意历史会话：

```
/session
```

会显示最近的会话列表，选择一个即可继续。

## 回退

做错了？回退到之前的消息：

```
/rewind
```

会显示最近的消息，选择一个回退点——之后的所有内容会被丢弃。

## 压缩

当对话上下文达到模型限制的 80% 时，omx 自动压缩会话：

1. 对话被总结
2. 提取关键记忆点
3. 新会话注入总结后开始

也可以手动触发压缩：

```
/compact
```

### 服务器压缩

默认在客户端压缩。如果需要，可在偏好设置中启用 **Server compaction**，让服务器处理。
